# Installation

Add this line to your application's Gemfile:

```ruby
gem 'evil_events'
```

And then execute:

```shell
$ bundle install
```

Or install it yourself as:

```shell
$ gem install evil_events
```

After all:

```ruby
require 'evil_events'
```
